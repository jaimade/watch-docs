# docwatch

Documentation decay detection system - monitors codebases to detect when documentation and code have drifted apart.

## Setup

```bash
# Create virtual environment
python3 -m venv venv

# Activate virtual environment
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt
```
